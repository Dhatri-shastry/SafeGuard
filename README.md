# SafeGuard – Smart Personal Safety System

SafeGuard is a personal safety application designed to help users respond quickly during emergencies. The system provides features like SOS alerts, real-time location tracking, voice-triggered emergency activation, shake alerts, fake calls, and quick access to emergency helplines and nearby help centers.

## 🚀 Features

- 🚨 SOS Emergency Alert – Send instant alerts to trusted contacts
- 📍 Live Location Tracking – Share real-time location during emergencies
- 🎙 Voice SOS Activation – Trigger emergency alerts using voice command
- 📳 Shake Alert – Activate SOS by shaking the phone
- 📞 Fake Call Feature – Simulate an incoming call for safety situations
- 🧭 Nearby Help Centers – Locate nearby hospitals and police stations
- ☎ Emergency Helplines – Quick access to important emergency numbers

## 🛠 Tech Stack

- React.js  
- JavaScript  
- HTML5  
- CSS3  
- Geolocation API  

## ⚙ Installation & Setup

Clone the repository

```bash
git clone https://github.com/Dhatri-Shastry/safeguard.git
```

Navigate to the project folder

```bash
cd safeguard
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm start
```

Open the browser and visit:

```
http://localhost:3000
```

## 🎯 Purpose of the Project

The goal of SafeGuard is to provide quick emergency assistance through real-time location sharing and safety features that help users stay protected in critical situations.
